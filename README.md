# Blind_man_Stick
The Blind Man Smart Stick is an assistive technology project built using an Arduino Nano microcontroller, an HC-SR04 ultrasonic distance sensor, and an active buzzer. The system is designed to help visually impaired individuals navigate their surroundings safely.

When the ultrasonic sensor detects an obstacle within a predefined distance (100 cm by default), the buzzer plays the Morse code sequence for 'ALRT' — acting as a distinctive audio alert that informs the user of a nearby obstacle. The pattern repeats as long as the obstacle remains in range.
